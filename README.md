# react-from-scratch

This project consists of:

1. React
2. GraphQL -
   Apollo Client
   Apollo Server
3. Reach Router
4. Material UI

To run the project: <br />
`git clone https://github.com/gbsimon87/statsy.git` <br />
`cd statsy` <br />
`yarn` (or `npm install`)<br />
`yarn run dev` (or `npm run dev`) <br />
